HTML tutorial

What is HTML ?

• HTML stands for Hyper Text Markup Language.
• It is the standard markup language for creating Web pages.
• It describes the structure of the web page.

1- Fundemintal tags :-

• HTML is a markup language that uses tags to define elements on a web page.
• The basic structure of an HTML document includes the following tags:
• <!DOCTYPE html> - This declaration defines the document type and version of HTML being used.
• <html> - This tag is the root element of an HTML page.
• <head> - This tag contains meta-information about the document, such as the title and links to stylesheets.
• <title> - This tag defines the title of the document, which appears in the browser's title bar or tab.
• <body> - This tag contains the content of the document, including text, images, and other elements.

2- Basic HTML Elements :-

• HTML provides a variety of elements to structure and format content on a web page.
• Some common HTML elements include:
• <h1> to <h6> - These tags define headings, with <h1> being the largest and <h6> being the smallest.
• <p> - This tag defines a paragraph of text.
• <a href="https://www.example.com"> - This tag creates a hyperlink to another web page or resource.
• <img src="image.jpg" alt="description"> - This tag embeds an image in the document, with a source URL and alternative text.
• <ul>, <ol>, and <li> - These tags create unordered (bulleted) and ordered (numbered) lists.

3- HTML Attributes :-

• HTML elements can have attributes that provide additional information about the element.
• Attributes are specified within the opening tag and usually come in name/value pairs.
• For example, in the <a href="https://www.example.com"> tag, "href" is an attribute that specifies the URL of the link.

4- HTML Forms :-

• HTML forms are used to collect user input.
• They consist of various input elements, such as text fields, checkboxes, radio buttons, and submit buttons.
• The <form> tag is used to create a form, and each input element is defined using its respective tag (e.g., <input> , <select> , etc.).

5- HTML Semantic Elements :-

• Semantic elements provide meaning to the content they contain.
• They help search engines and browsers understand the structure of the page.
• Examples include <header> , <nav> , <article> , <section> , and <footer> .
